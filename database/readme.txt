Versions are split as following:

	Major changes, change the Integer of the version (1.0 > 2.0 > 3.0 etc..)
	Minor changes, change the Fractional of the version (1.0 > 1.1 > 1.2 etc..)

Only major changes are subcatagorized.

Documentation & Code versions are always the latest minor update

	This includes but is not limited to: 
		SQL Create Script.
		SQL Insert Script.
		Attribute description.
